https://github.com/Xaliss/Synchro-Vim.git

Thanks to Neil !
I can work anywhere !
Just do the following:

Create symlinks:

    ln -s ~/.vim/vimrc ~/.vimrc
    ln -s ~/.vim/gvimrc ~/.gvimrc

Switch to the `~/.vim` directory, and fetch submodules:

    cd ~/.vim
    git submodule init
    git submodule update
